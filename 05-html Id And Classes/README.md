## ğŸ”¹ Class

A **class** is used to apply the same CSS styles to **multiple HTML elements**.  
It is reusable and defined using a dot (`.`) in CSS.

### Example:

```html
 <div class="red bg-yellow">First</div> 
```
```css

.red {
    background-color: aquamarine;
}

.bg-yellow {
    color: blueviolet;
}
```

## ğŸ”¹ ID

An id is used to uniquely identify one specific HTML element on a webpage.
Each id must be unique and is defined using a hash (#) in CSS.

### Example:
```html
<div id="seconddiv">Second</div>
```
```css
#seconddiv {
    background-color: blueviolet;
}
```

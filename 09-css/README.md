### ğŸ”¹ Internal CSS
**Internal CSS** is written inside the `<style>` tag in the `<head>` section of an HTML document.  
It is useful when styles are applied to **a single page only**.

**Example:**
```html
<style>
    div {
        color: rgba(1, 248, 63, 0.671);
        background-color: black;
    }
</style>
```

# 🎨 CSS Filters Practice Project

This project demonstrates how to use **CSS Filters** to visually modify images directly using CSS properties.

The example applies different filter effects such as blur, brightness, contrast, grayscale, invert, and sepia on an image.

---

## 📁 Files Included

- **filters.html** – Demonstrates different CSS filter properties using internal CSS.
- **gokuimage.webp** – Image used for applying filter effects.

---

# 📌 What Are CSS Filters?

CSS Filters allow you to apply graphical effects like:

- Blur
- Brightness
- Contrast
- Grayscale
- Invert
- Sepia

These effects are applied directly through CSS without editing the original image.

---

# 💻 Important CSS Used

```css
img{
    filter: blur(2px);

    /* filter: brightness(70%); */
    /* filter: contrast(250%); */
    /* filter: grayscale(100%); */
    /* filter: invert(100%); */
    /* filter: sepia(90%); */
}
```

---

# 🔎 Explanation of Each Filter

### 🔹 blur(2px)
Blurs the image by 2 pixels.

### 🔹 brightness(70%)
Reduces brightness to 70% of original.

### 🔹 contrast(250%)
Increases contrast to 250%.

### 🔹 grayscale(100%)
Converts image completely to black and white.

### 🔹 invert(100%)
Inverts the colors of the image.

### 🔹 sepia(90%)
Applies a brown vintage effect.

---

# 🔥 You Can Also Combine Filters

Example:

```css
filter: blur(2px) brightness(120%) contrast(150%);
```

Multiple filters can be applied together in a single line.

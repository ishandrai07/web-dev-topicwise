# 🎬 Netflix Landing Page Clone

This project is a **Netflix-inspired landing page** built using **HTML, CSS, and JavaScript**.  
It recreates the main hero section and a **Trending Movies slider** similar to the Netflix homepage.

## 📁 Files Included
- **index.html** – Main structure of the webpage
- **style.css** – Styling and layout of the page
- **script.js** – JavaScript for the movie slider functionality

## ✨ Features
- Netflix-style hero section with background image
- Navigation bar with logo and sign-in button
- Email input and "Get Started" button
- Trending movies section
- Horizontal scrolling movie cards
- Left and right slider buttons

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 Learning Outcomes
- Building a real-world UI clone
- Creating responsive layouts
- Implementing horizontal scroll sliders using JavaScript
- Practicing modern CSS styling
